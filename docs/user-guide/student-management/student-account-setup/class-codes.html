Enabling and using Class Codes

You can allow students to join your school account and/or enroll in your class using class codes. To enable this option by clicking the Classes tab and selecting a specific class. The class code will be displayed on the right side of the overview section.

  1. Go to the Classes tab and choose the class you want students to be bale to join.
  2. On the right side of the screen, check the box next to Students can join this class.
  3. Share the class code or the URL with students.


From the student interface, there will be a + Join a class option on the homepage. Once clicked, the student will be able to enter the class code in order to be added to the class.
