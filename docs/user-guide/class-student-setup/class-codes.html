Class Codes

TypingClub now allows students to both join your school account and enroll in a specific class using Class Codes. Class Codes can be activated on a per-class basis.


Activating Class Codes

  1. Go to the Classes tab and choose the class you want students to be able to join.
  2. On the right side of the screen, check the box next to Students can join this class.
  3. Share the class code or the URL with students.


Notice: Please note that if you duplicate a class, this will create a new, unique class code.

Student view of joining a class

From the student interface, there will be a + Join a class option on the homepage. Once clicked, the student will be able to enter the class code in order to be added to the class.





Student view of joining an account

Have students follow these steps:

  1. Go to your school's unique TypingClub URL (i.e. school.typingclub.com).
  2. Click Enter a Class Code.


  3. Enter the class code and click Sign Up.


  4. Students can either sign up with Google, Office 365, or by entering their details into the form.


  5. Students will be automatically logged into their new account and enrolled in their class.
