How do I install the Chrome extension?

To streamline the sign in process, you can install the EdClub Chrome extension. After a one-time setup process, the extension will direct your students to the correct portal URL.

Notice: The extension to EdClub is not be searchable publicly from the Google App Store. To install it, you must go to the following URL: EdClub Extension

This extension will guide your students through the process of finding your school's portal URL and show them how to use it to sign in to EdClub.
