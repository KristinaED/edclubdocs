How can I adjust the minimum star requirement?

  1. Go to the Classes tab
  2. Click the class name to access the class overview
  3. Click the Student Settings link
  4. Choose the number of stars you'd like students to earn in order to move forward from the dropdown menu
  5. Scroll down to the bottom of the page and update your settings before navigating away.

The preferences chosen from the teacher portal will be set for the entire class. At the moment, this setting cannot be customized for individual students.
