How can I change daily and weekly time goals?


Changing time goals for one class:

  1. Go to the Classes tab and click the name of class you'd like to update
  2. Click the ''Student Settings" link
  3. Use the drop down menu to select the desired daily and/or weekly time goals
  4. Scroll to the bottom of the page and click "Update Settings" to save your changes

Changing time goals for multiple classes at once:

  1. Go to the Classes tab
  2. Check the box next to the classes you'd like to update
  3. Click the Bulk Update button
  4. Click "Goal for Daily Practice Time" or "Goal for Weekly Practice Time" to see the available options. Select your preference from the drop-down menu
  5. Click "Bulk Update" to save your changes
