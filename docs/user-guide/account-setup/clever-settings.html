Clever Status & Settings

This section of your account settings will allow you to view the status of your sync, logs of past syncs, and change the way we handle your data. Please note that these settings are only accessible to Account Administrators.

  • Overview

    The Overview tab displays your sync status brokendown by type of data: schools, teachers, classes, students, school admins, and admins.

    1. Go to the Settings tab.
    2. Scroll down and click Clever.
    3. Click Update. After a few moments, the sync status will refresh.


  • Logs

    The Logs tab allows you to view both an on-screen summary of what was created, updated and deleted during each sync, as well as a detailed list.

    1. Go to the Settings tab.
    2. Scroll down and click Clever.
    3. Click the Logs tab.



  • Settings

    The Clever Settings tab allows you to update your sync options and run a sync at any point during the day.

    1. Go to the Settings tab.
    2. Scroll down and click Clever.
    3. Click the Settings tab.

    Sync Options

    TypingClub allows you to choose alternate setups for the following fields:

    • Student and/or teacher passwords
      • Leave passwords alone (default)
      • Make passwords blank
      • Custom password for everyone
      • Randomly generated password
      • Use "username"
      • Use "first name"
      • Use "last name"
      • Use "SIS ID"
      • Use "student number"
    • Student usernames
      • Leave usernames alone
      • Make usernames blank
      • Use "username"(default)
      • Use "SIS ID"
      • Use "student number"
      • Use the first part of email (before the @ symbol)
    • Decide if we sync email addresses
    • Change the way classes are named
      • You can combine fields to create a different name structure.
    • Change the way student IDs are listed
      • Use "student number"(default)
      • Make student ID blank
      • Combine "school number" and "student ID"

    Run a sync

    TypingClub pulls in new data from Clever each night around 2am ET. If you want to sync your account at any other point, you can do so here. There are two options:

    • Sync Up
        This is an incremental sync that will bring in any changes from Clever that have occurred since the last sync.
    • Fresh Sync
        Normally, Clever syncs pick up only changes that occurred since the last sync. If you wish to start a fresh sync that pulls all the data, use this option.
      Warning: Please be warned that this option will override any other manual changes you may have made to individual records.
