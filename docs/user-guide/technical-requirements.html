What are TypingClub's technical requirements?


TypingClub is 100% web-based so there are no downloads or plugins required. All you need is one of the following up-to-date browsers: Google Chrome, Firefox, Safari, or Internet Explorer. We recommend using the latest version of Google Chrome for the best experience.

You may want to add these URLs to your firewall whitelist:

  • Your account's portal URL: https://schoolname.typingclub.com
  • Our media site: https://static.typingclub.com
