How can I activate placement tests for Typing Jungle?

Teachers can utilize placement tests to assess where students' current touch typing abilities are. A placement test can be assigned to a class by following these steps:


  1. Select the Classes tab on the left side of your dashboard
  2. Select your class
  3. Click the Lesson Plans link
  4. For each lesson plan, you will see a column labeled Placement which will allow you to enable/disable a placement test
  5. Click Update to save your changes

Keep in mind that placement tests are available only on updated versions of Typing Jungle. To check if your copy is up to date, go to:
Lesson Plans > Typing Jungle.

If an update is available, you'll be prompted to update your Typing Jungle. Then you can proceed to enable the placement test for your class.

To view a student's placement test score, you will first select the specific student, go to their activity attempts page and click on Placement tests.
