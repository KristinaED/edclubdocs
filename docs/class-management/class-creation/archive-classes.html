Archive classes

Teachers have the ability to pause classes by archiving them. This is helpful when teachers want to pause the class but still preserve the roster, the class settings, and lesson plans assigned. When you disable a class, students will not be able to access the lesson plans that are assigned to that class. You can use this feature to either hide certain lesson plans from students or to preserve a roster for a later date.


How to disable classes:

  1. Go to the Classes tab.
  2. Check the box for the class(es) you'd like to disable.
  3. Click the Disable button at the top of the list of classes.

How to view disabled classes:

  1. Go to the Classes tab.
  2. Click the Show Filters button on the right side of the screen.
  3. Under Classes that are, select either Inactive or Both active and inactive.

How to activate previously disabled classes:

  1. Go to the Classes tab.
  2. Click the Show Filters button on the right side of the screen.
  3. Under Classes that are, select either Inactive or Both active and inactive.
  4. Check the box next to the class(es) you'd like to activate.
  5. Click the Activate button at the top of the list of classes.
Warning: Disabled classes are hidden from your class list. You must use the filters to view inactive classes.
