Activating placement test in Typing Jungle

Teachers can utilize a pre-made placement test in Typing Jungle to assess where students' current touch typing abilities are. The placement test will unlock skills that students are already proficient in, however, students will not receive credit for the unlocked lessons. The Typing Jungle placement test can be assigned to a class by following these steps:


  1. Click the Classes tab and select your class.
  2. Click Lesson Plans. For each lesson plan, you will see a column labeled Placement which will allow you to enable/disable a placement test.
  3. Click Update to save your changes.
Notice: Keep in mind that placement tests are available only on updated versions of Typing Jungle. To check if your copy is up to date, go to Lesson Plans Typing Jungle. If an update is available, you'll be prompted to update your Typing Jungle. Then you can proceed to enable the placement test for your class.

How do I view placement test results?

To view a student's placement test score, you will first select the specific student, go to their activity attempts page and click Placement tests. A class-wide placement test report is coming soon.
