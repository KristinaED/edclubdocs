Creating placement tests in custom lesson plans

Teachers have the ability to create placement tests in their custom lesson plans. Once created and assigned, the placement test will unlock skills that students are already proficient in. Remember that students will not receive credit for the unlocked lessons.

Step 1: Create custom placement test

Placement tests can be added to a lesson plan by following these steps:

  1. Click the Lesson Plans tab.
  2. Select the lesson plan.
  3. Click the plus sign to add a new lesson
  4. Select a lesson type
  5. Type your desired text and under the Placement Test section, click Placement Test.
  6. When you're finished, save the lesson by clicking Done.

Step 2: Assign custom placement test to your class

Once the placement test has been created, you will go back to the specific class and enable it:

  1. Click the Classes tab and select your class.
  2. Click Lesson Plans and find the custom lesson plan. You will see a column labeled Placement which will allow you to enable/disable the placement test you just created.
  3. Click Update to save your changes.

How do I view placement test results?

To view a student's placement test score, you will first select the specific student, go to their activity attempts page and click Placement tests. A class-wide placement test report is coming soon.
