Managing the backspace behavior

Teachers can decide whether or not to allow students to use the backspace to correct their mistakes. The default class settings allow the use of backspace, however, they can be changed by following the next steps:

  1. Go to the Classes tab.
  2. Open the class you'd like to edit.
  3. Click the Edit Class tab.
  4. Scroll down and select your preferred backspace behavior.
  5. Update your settings on the bottom of the page.
