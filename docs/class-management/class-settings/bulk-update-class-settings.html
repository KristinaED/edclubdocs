Bulk-update class settings

Many of the class settings available on TypingClub can be updated in bulk. You can update the following settings for multiple classes at once:

  • Scoreboard visibility
  • Jumping ahead
  • Assign lesson Plan
  • Un-assign lesson plan
  • Goal for daily practice time
  • Goal for weekly practice time
  • Which day your week begins

How to update multiple classes at once:

  1. Go to the Classes tab.
  2. Check the box next to the classes you'd like to update.
  3. Click the Bulk Update button.
  4. Adjust the desired settings using the pop-up menu. Click each setting to see the available options.
  5. Click Bulk Update to save your changes.
