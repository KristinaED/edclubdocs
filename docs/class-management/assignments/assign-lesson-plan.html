Assigning lesson plans

Every TypingClub class has at least one lesson plan assigned, typically the default lesson plan Typing Jungle. However, teachers can assign as many lesson plans as they like. Learn how to assign lesson plans below:


  1. Click the Classes tab.
  2. Choose the class of interest.
  3. Select Lesson Plans to see the lesson plans assigned to this class.
  4. Click the +Assign lesson plan to this class button and you will see a list of lesson plans which can be assigned to the class. The lesson plans that appear in this list are lesson plans that have been installed in your account. If you'd like to browse the Lesson Plan Library and possibly install new lesson plans, go to Lesson Plans Lesson Plan Library. Keep in mind that all lesson plans are available for download at no extra charge.
  5. When you're done selecting the lesson plan you wish to assign, click Add Lesson Plans.
